# Navegacao-ReactNative

Repositório de estudos sobre navegação em aplicações React Native.

## Sobre o projeto

O **Navegacao-ReactNative** reúne práticas e exemplos desenvolvidos durante meus estudos sobre navegação em aplicativos mobile.

O objetivo foi entender diferentes formas de navegação entre telas, organização de rotas e uso de ícones em aplicações React Native.

Este repositório é importante porque navegação é uma parte essencial no desenvolvimento de aplicativos reais.

## Conteúdos praticados

- Navegação entre telas
- Stack Navigation
- Tab Navigation
- Drawer Navigation
- Uso de ícones
- Organização de rotas
- Criação de telas
- Prática com estrutura de navegação em React Native

## Tecnologias utilizadas

- React Native
- JavaScript
- Expo
- React Navigation
- Node.js / npm

## Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/JohnElthon/Navegacao-ReactNative.git
```

2. Acesse a pasta do exemplo que deseja executar.

3. Instale as dependências:

```bash
npm install
```

4. Execute o projeto:

```bash
npm start
```

5. Abra no emulador Android/iOS ou no aplicativo Expo Go.

## Observação

Este repositório foi desenvolvido como material de estudo para praticar diferentes formas de navegação em React Native. Algumas pastas podem representar exemplos separados de aula.
